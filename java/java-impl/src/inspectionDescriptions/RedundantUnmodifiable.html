Reports redundant calls to unmodifiable collection factories within the Collections class. If the argument that is passed to an unmodifiable collection factory is already immutable, such wrapping becomes pointless.

Example:

Collections.unmodifiableList(Collections.singletonList("abc"));
In order to detect the methods that return unmodifiable collections, the inspection uses the @Unmodifiable annotation. Use this annotation if you want to extend the inspection to your own unmodifiable collection factories.

New in 2020.3
